가설검정 기본 이론

작업형3의 모든 문제가 이 틀 위에 있습니다.

귀무가설(H₀) vs 대립가설(H₁).
  • H₀: "차이/효과/관계가 없다" (기존 통념, 기본 가정)
  • H₁: "차이/효과/관계가 있다" (입증하고 싶은 주장)
p-value. H₀가 참이라고 가정했을 때, 관측된 것만큼 극단적인 결과가 나올 확률. p가 작으면 "H₀ 가정으로는 이 데이터가 거의 안 나온다" → H₀를 기각.
p < 0.05 H₀ 기각 → 대립가설 채택
"유의한 차이/효과/관계가 있다"
p ≥ 0.05 H₀ 기각 못함 (채택)
"유의한 차이가 없다"
단측/양측 (alternative). 'two-sided'(≠, 기본), 'greater'(첫 인자가 더 큼), 'less'(첫 인자가 더 작음). 문제에서 "더 크다/작다/감소"가 나오면 단측 검정.

ch1 · t-검정 (가설검정)

평균 비교. 정규성·등분산 전제 확인 → 위반 시 비모수 대안.

세 가지 t-검정

검정함수용도H₀
단일표본ttest_1samp(x, μ₀)표본 평균 vs 기준값μ = μ₀
대응표본ttest_rel(a, b)같은 대상 전·후 비교μ_diff = 0
독립표본ttest_ind(a, b)두 독립 집단 평균 비교μ₁ = μ₂

전제 검정 (먼저 확인)

정규성 — Shapiro-Wilk

from scipy import stats
stat, p = stats.shapiro(x)
# H0: 정규분포를 따른다
# p < 0.05 → 정규성 위반 → 비모수 검정

등분산 — Levene

stat, p = stats.levene(a, b)
# H0: 두 집단 분산이 같다
# p < 0.05 → ttest_ind(equal_var=False) (Welch)

비모수 대안 (정규성 위반 시)

모수 검정→ 비모수 대안함수
대응표본 t윌콕슨 부호순위wilcoxon(a, b)
독립표본 t맨-휘트니 Umannwhitneyu(a, b)

코드 패턴

from scipy import stats

# 단일표본: 평균이 95보다 작은가? (단측)
stat, p = stats.ttest_1samp(df['x'], 95, alternative='less')

# 독립표본: 등분산 확인 후
stat, p = stats.levene(a, b)
stat, p = stats.ttest_ind(a, b, equal_var=True, alternative='two-sided')

# 대응표본: 차이 컬럼 활용
df['diff'] = df['after'] - df['before']
stat, p = stats.ttest_rel(df['after'], df['before'], alternative='greater')

print(round(stat, 3), round(p, 4))
실전 흐름. ① shapiro로 정규성 → ② 독립표본이면 levene로 등분산 → ③ 만족하면 t-검정, 위반하면 비모수.

ch2 · 분산분석 (ANOVA)

3집단 이상의 평균을 동시에 비교. t-검정을 여러 번 하면 오류가 누적되므로 ANOVA 사용.

이론. ANOVA는 "집단 간 분산 / 집단 내 분산" = F비로 검정합니다. 집단 간 변동이 집단 내 변동보다 충분히 크면 평균이 다르다고 판단. 전제: 정규성, 등분산, 독립성.

일원 분산분석 (One-way) — 요인 1개

H₀: 모든 집단 평균이 같다 (μ₁ = μ₂ = … = μ_k) · H₁: 적어도 하나는 다르다
# 방법1) scipy 간편
from scipy import stats
stat, p = stats.f_oneway(A, B, C, D)

# 방법2) statsmodels (회귀 틀)
from statsmodels.formula.api import ols
from statsmodels.stats.anova import anova_lm
model = ols('성장 ~ C(비료)', data=df).fit()
print(anova_lm(model))   # F, PR(>F) 확인
C(비료)의 C()는 범주형 처리 지시자. PR(>F)가 p-value입니다.

이원 분산분석 (Two-way) — 요인 2개 + 상호작용

model = ols('수확량 ~ C(비료) * C(물주기)', data=df).fit()
print(sm.stats.anova_lm(model))
# C(비료):C(물주기) 행 = 상호작용 효과
항의미H₀
C(요인A)요인A 주효과A는 평균에 영향 없음
C(요인B)요인B 주효과B는 평균에 영향 없음
C(A):C(B)상호작용A 효과가 B에 따라 달라지지 않음

ch3 · 카이제곱검정 (χ²)

범주형 데이터의 빈도(count)를 분석. χ² = Σ(관찰−기대)²/기대.

세 가지 카이제곱

검정함수용도
적합도chisquare(관찰, 기대)관찰 빈도가 특정 분포(비율)를 따르는가
독립성chi2_contingency(표)두 범주형 변수가 독립인가
동질성chi2_contingency(표)여러 집단의 분포가 동일한가
참고. 독립성과 동질성은 수학적으로 같은 함수(chi2_contingency)를 쓰며, 연구 설계에 따라 해석만 달라집니다.

① 적합도 검정 (Goodness-of-fit)

H₀: 관찰빈도가 기대 비율을 따른다
from scipy.stats import chisquare
n = 300
observed = [150, 120, 30]
expected = [0.50*n, 0.35*n, 0.15*n]   # 기대빈도 = 비율 × 전체
stat, p = chisquare(observed, expected)

② 독립성 / 동질성 검정

H₀(독립성): 두 변수는 독립이다 (관계 없다)
from scipy.stats import chi2_contingency
import pandas as pd

# 원자료 → 교차표
table = pd.crosstab(df['성별'], df['운동'])
stat, p, dof, expected = chi2_contingency(table)
print(round(stat, 3), round(p, 4), dof)
# expected = 기대빈도 (각 셀 ≥ 5 권장)
주의. 반환값이 4개(통계량, p값, 자유도, 기대빈도). 기대빈도가 5 미만 셀이 많으면 카이제곱 신뢰도가 떨어집니다.

ch4 · 선형회귀 & 상관

연속형 변수 간 관계를 모델링. statsmodels의 OLS와 formula API 사용.

상관분석

방법함수특징
피어슨pearsonr(x, y), df.corr()선형 관계, 정규성 가정
스피어만spearmanr(x, y)순위 기반, 단조 관계
켄달kendalltau(x, y)순위 일치도, 이상치 강건
 
from scipy import stats
r, p = stats.pearsonr(df['키'], df['몸무게'])
# r: -1~1 상관계수, p: H0(ρ=0) 검정

OLS 선형회귀

이론. 최소제곱법(OLS)으로 Y = β₀ + β₁X₁ + … 의 계수를 추정. 각 계수의 p-value는 "이 변수가 Y에 선형 영향을 주는가(H₀: β=0)"를 검정.
from statsmodels.formula.api import ols

model = ols('매출액 ~ 광고비 + 직원수', data=df).fit()
print(model.summary())

model.params['광고비']        # 회귀계수
model.pvalues['광고비']       # p-value (β=0 검정)
model.rsquared                # R² (설명력)
model.predict(new_df)         # 새 데이터 예측
model.conf_int(alpha=0.05)    # 95% 신뢰구간
sum(model.pvalues[1:] < 0.05) # 유의 변수 수 (절편 제외)
범주형 독립변수. ols('score ~ hours + material_type', ...) — 문자열 컬럼은 자동으로 더미 변수화(material_type[T.도서] 형태, 첫 범주가 기준).

ch5 · 로지스틱 회귀

종속변수가 이진(0/1)일 때. 로그-오즈를 선형 모델링하고, 오즈비로 해석.

이론. 로지스틱 회귀는 확률 p를 직접 모델링하는 대신 로그-오즈 ln(p/(1−p)) = β₀ + β₁X₁ + … 를 선형으로 적합합니다. 예측값은 0~1 확률.

모델 적합 & 핵심 출력

from statsmodels.formula.api import logit
import numpy as np

model = logit('disease ~ age + bmi + smoker', data=df).fit()
print(model.summary())

model.params['bmi']                  # 회귀계수 (로그-오즈 변화)
model.pvalues['bmi']                 # 유의성
np.exp(model.params['bmi'])          # 오즈비 (OR)
np.exp(model.params['bmi'] * 3)      # bmi 3단위 증가 시 OR
model.llf                            # 로그우도
-2 * model.llf                       # 이탈도(deviance)

오즈비(Odds Ratio) 해석

OR = exp(β). 독립변수 1단위 증가 시 오즈가 몇 배 되는지.
  • OR > 1 : 사건 발생 가능성 증가
  • OR < 1 : 사건 발생 가능성 감소
  • OR = 1 : 영향 없음

예측 & 분류 성능

# 확률 예측 → 임계값으로 분류
proba = model.predict(test)
pred = (proba > 0.5).astype(int)     # 임계값 0.4, 0.3 등 문제 조건

from sklearn.metrics import accuracy_score, recall_score
accuracy_score(y_true, pred)         # 정확도
recall_score(y_true, pred)           # 민감도(재현율)
1 - accuracy_score(y_true, pred)     # 오류율

# 가장 위험한(확률 높은) 케이스
proba.idxmax()
기준 범주 지정. C(lifestyle, Treatment("S")) — 'S'를 기준으로 더미 비교.

검정 선택 가이드

"무슨 검정을 써야 하지?"에 대한 빠른 답.

상황검정함수
연속형 1개 vs 기준값단일표본 tttest_1samp
연속형, 짝지어진 전·후대응표본 t / 윌콕슨ttest_rel / wilcoxon
연속형, 독립 두 집단독립표본 t / 맨휘트니ttest_ind / mannwhitneyu
3집단 이상 평균일원 ANOVAf_oneway / anova_lm
요인 2개의 효과이원 ANOVAols(...).anova_lm
관찰빈도 vs 기대분포적합도 카이제곱chisquare
두 범주형 변수 관계독립성 카이제곱chi2_contingency
연속 Y ~ 연속 X선형회귀 / 상관ols / pearsonr
이진 Y ~ 예측변수로지스틱 회귀logit

전제 확인 순서 (실전 루틴)

  1. t-검정: shapiro(정규성) → 위반 시 비모수
  2. 독립표본 t: levene(등분산) → equal_var 결정
  3. ANOVA: 집단별 shapiro + levene → f_oneway
  4. 카이제곱: 기대빈도 ≥ 5 셀 확인

p-value 판정 한 줄 정리

p < 0.05 귀무가설 기각 · "유의하다"
p ≥ 0.05 귀무가설 기각 못함 · "유의하지 않다"
