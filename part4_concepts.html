작업형1 데이터 분석 (pandas)

단일 CSV에서 파생변수 → 집계 → 필터 → 최종 수치 1개 출력

표준 풀이 흐름

read_csv→ 형변환 / 파생변수→ groupby / quantile / corr→ idxmax / nlargest / iloc→ round / int / print

핵심 개념별 함수

데이터 불러오기 & 형변환

  • pd.read_csv()
  • pd.to_datetime() — 날짜형 변환
  • .dt.year, .dt.month, .dt.to_period('M')
  • .dt.total_seconds() — 시간 차이(초→분)
  • .astype() — 자료형 변환

문자열 처리 (.str)

  • .str[:n] — 앞 n글자 슬라이싱 (연도 추출)
  • .str.contains('C') — 포함 여부
  • .str.split(), .str.len() — 단어 수
  • .str.count(' ') + 1 — 단어 수 (대안)

파생변수 & 행 연산

  • 컬럼 간 사칙연산 (quantity * price)
  • df.iloc[:, start:end].sum(axis=1) — 행 합계
  • 비율 계산: (a + b) / c
  • CAGR: (마지막/처음) ** (1/n) - 1
  • 조건부 값 변경: df.loc[cond, col] = value

필터링 & 조건

  • boolean mask: cond1 & cond2
  • 구간 조건: (x > 0.4) & (x < 0.5)
  • len(df[cond]) — 조건 만족 행 수
  • df[cond] — 조건 필터

집계 & 그룹

  • .groupby().mean(), .sum()
  • .transform('mean') — 그룹별 평균으로 결측 대체
  • .value_counts() — 빈도
  • pivot_table(), melt() — wide ↔ long
  • .unstack() — 피벗 후 변환

통계 & 순위

  • .quantile(0.25), .quantile(0.75) — IQR
  • IQR 이상치: Q1 − 1.5×IQR, Q3 + 1.5×IQR
  • .corr() — 상관계수 행렬
  • .idxmax(), .nlargest(n)
  • .sort_values(), .iloc[n]
  • .mean(), .median(), .std()

결측치 & 중복

  • .isnull().sum() — 결측 개수
  • .fillna(median()) — 중앙값 대체
  • .dropna() — 결측 행 제거
  • .drop_duplicates() — 중복 제거

스케일링 (sklearn)

  • from sklearn.preprocessing import StandardScaler
  • StandardScaler().fit_transform() — ch7
  • from sklearn.preprocessing import MinMaxScaler
  • MinMaxScaler().fit_transform() — ch8
  • 스케일링 후 .std() 차이 계산

자주 나오는 3단 패턴

  1. 파생변수 만들기 — BMI, 학생/교사 비율, 총 범죄 건수, order_amt 등
  2. 그룹 집계 — groupby, pivot, 연도별/카테고리별 합계
  3. 최댓값·순위 행 추출 — idxmax, nlargest, sort_values + iloc

from statsmodels.formula.api import logit
import numpy as np
from sklearn.metrics import accuracy_score, recall_score

# 1) 모델 적합
model = logit('disease ~ age + bmi + smoker', data=train).fit()
print(model.summary())

# 2) 회귀계수 / 오즈비
print(round(model.params['bmi'], 3))            # 회귀계수
print(round(np.exp(model.params['bmi']), 3))     # 오즈비

# 3) 예측 → 임계값 분류 → 성능
proba = model.predict(test)
pred  = (proba > 0.5).astype(int)
print(round(accuracy_score(y_true, pred), 3))    # 정확도
print(round(recall_score(y_true, pred), 3))      # 민감도
